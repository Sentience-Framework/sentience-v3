# Stdio

Sentience offers an Stdio class to make writing to the console easier.

The Stdio class has the following methods:
```
Stdio::print()
Stdio::printLn()
Stdio::printF()
Stdio::printFLn()

Stdio::error()
Stdio::errorLn()
Stdio::errorF()
Stdio::errorFLn()
```

The only accepted input type is strings. The methods don't encode the provided data.
